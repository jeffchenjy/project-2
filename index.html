About Project

Firmware
STM32

M0開發板提供豐富的硬體接口和處理能力，控制與運用各類感測器收集環境數據。

Sensor

透過各種感測器收集環境數據，如溫度、濕度、距離等等。並與STM32連接

Raspberry Pi

使用Mjpg-streamer與Django分別提供網路Camera畫面與網路接收指令的伺服器端

APP
Android Studio

主要以Kotlin進行開發，介面設計以直覺性和美觀性為主，提升使用者體驗。

Code

透過使用Fragment增加APP效能，也便於維護和擴展。

Firebase

擁有使用者登入、註冊、個人資料顯示與修改介面，並透過Firebase管理使用者資料。

APP Features

Material Design

應用程式的主題樣式主要使用Material3的外觀介面。關於Material Design >>

Efficiency

透過使用Fragment來減少呼叫新的Activity頻率以增加APP效能。關於Fragment >>

RecyclerView

透過使用RecyclerView取代ListView來提升APP性能，如效能上升、靈活性更高、局部刷新、更好的記憶體管理等等。 關於RecyclerView >>

Flexibility

使用者可以自訂發送給自走小車的命令，以及Wi-Fi控制所連接的URL。

    Car Features

    STM32

    所使用的開發板為STM32F072RB(M0)，主要控制自走小車動作、驅動馬達、感測器、LCD等等。關於M0開發板 >>

    Wi-Fi

    透過Raspberry Pi執行Mjpg-streamer與Django分別提供網路Camera畫面與網路接收指令的伺服器端，以便控制車子。 關於Mjpg-streamer >>

    Sensor and function

    使用各種不同的感測器與功能模組，如超音波(HC-SR04)、紅外線避障(GP2Y0A21YK0F、TCRT 5000)、溫溼度(DHT22)、DFPlayer mini等等。
