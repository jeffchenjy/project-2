About Project

APP
Android Studio

主要以Kotlin進行開發，介面設計以直覺性和美觀性為主，提升使用者體驗。

Code

透過使用Fragment增加APP效能，也便於維護和擴展。

API

使用retrofit透過API與Django進行資料的傳遞，並擁有使用者登入、註冊、個人資料顯示與修改介面。

Server
Django

使用Django來管理資料庫的資料與提供API與GET、POST網址進行資料的傳遞。

Security

使用Django REST framework來建立API網址，並使用其提供的TokenAuthentication來保護網站資料。

SQL

最初使用MySQL建立資料庫，之後利用PostgreSQL來建立資料庫。

APP Features

Material Design

應用程式的主題樣式主要使用Material3的外觀介面。關於Material Design >>

Efficiency

透過使用Fragment來減少呼叫新的Activity頻率以增加APP效能。關於Fragment >>

RecyclerView

透過使用RecyclerView取代ListView來提升APP性能，如效能上升、靈活性更高、局部刷新、更好的記憶體管理等等。 關於RecyclerView >>

Use an API

APP主要是透過retrofit2來進行 HTTP 請求，其有許多的優點，如簡潔的 API 設計、內置的線程管理、支持多種編解碼器、強大的請求和響應處理、支持異步和同步請求、易於擴展。 關於retrofit2 >>

More function

有更多的小功能提升使用者體驗，如EditText輸入偵測、animate()產生動圖、轉場動畫等等。

Django

使用Django來管理會員資料、會員頭像...等等。

Server Features

Django

使用Djnago作為伺服器端，Django 是一個用 Python 語言開發的高級 Web 應用框架， 它使用了 MTV架構的設計模式，旨在幫助開發人員快速地構建高效、安全且可擴展的 Web 應用程序。關於Django >>

API

使用Django REST framework來建立API網址。它使得在 Django 中創建 Web API 變得簡單且高效，同時提供了許多功能和工具，讓開發人員可以輕鬆地創建功能完整、安全、易於測試和文檔完善的 API。 關於Django REST framework >>

PostgreSQL

PostgreSQL 是一個強大的開源關聯式數據庫管理系統（RDBMS），具有廣泛的功能和高度的擴展性。其優點如開源和免費、關聯式數據庫、 高度擴展性、完整的 ACID 支持、多版本並發控制、豐富的擴展功能。 關於PostgreSQL >>
